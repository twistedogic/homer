# Real Estate Investment Calculator

A simple, browser-based real estate investment calculator for Hong Kong properties.

## Features

- Calculate Internal Rate of Return (IRR), Net Present Value (NPV), and year-by-year cash flows
- 12 inputs including Hong Kong Property Tax (15% default)
- HKD currency formatting
- Mobile-responsive layout
- No dependencies — pure HTML, CSS, and JavaScript

## Usage

Open `index.html` in any modern browser.

## Deployment

This site is hosted on GitHub Pages.

To deploy your own version:

1. Fork this repository
2. Go to Settings → Pages
3. Select "Deploy from a branch" and choose `main` / `root`
4. Your calculator will be live at `https://yourusername.github.io/repo-name/`

## Assumptions

- 15% Hong Kong Property Tax (with 20% repairs allowance)
- 7% selling costs (agent fees + closing costs)
- No property taxes beyond input 12, no insurance, maintenance, or HOA
